Project Details

Led a team of 9 focusing on the deployment of a Web App within the Microsoft Azure cloud platform.

  • Developed a React.js application and implemented a CI/CD pipeline using Azure Web App services’ PaaS offerings.
  • Managed a Microsoft MySQL database for the application & created Power BI dashboards to provide actionable insights.
  • Employed a CI/CD pipeline & leveraged Azure ML Studio and Azure Web App services’ PaaS offerings, built on Azure Resource Manager.

Achieved first place in the project showcase that utilized an end-to-end Software, Devops and ML stack, earning recognition from both the CEO of Quadrant Technologies, and the Vice President of Microsoft Database.
