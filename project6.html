Project Description

This project focuses on developing and deploying advanced machine learning and deep learning models from scratch. The models are designed to address complex real-world problems with high accuracy and efficiency.

Course Projects:

  • Course 1: EE 511 - Advanced Introduction to Machine Learning
  • Course 2: CSE 599 - Advanced Deep Learning

Project Highlights:

  • Developed a range of machine learning models using supervised and unsupervised learning techniques.
  • Designed and implemented deep learning models for image and text data processing.
  • Experimented with various architectures, including CNNs, RNNs, and Transformers.
  • Fine-tuned pre-trained models to achieve higher accuracy on specific tasks.
